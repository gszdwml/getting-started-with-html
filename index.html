HTML Attributes

Attributes attach a function to a word or group of words in the HTML document. For example, HREF specifys the hyperlink of the anchor tag.

Images


Image tags do not require a closing tag but do require an alternate or "alt". This is so the browser can have a default label or text if the image is not located.

Whitespace

This tag helps with spacing and page layout. For example the br tag breaks the currrent line and moves it to the next one.
The p tag is like a box or container that text fits into and is contained to that box; Like a paragraph.
